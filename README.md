# Expo Judge Pro

A mobile-first engineering expo judging system.

**Live app:** https://YOUR-USERNAME.github.io/expo-judge/

## Features
- Score stalls across 5 criteria (Innovation, Practical Use, Technical Quality, Presentation, Future Scope)
- Auto-ranking and leaderboard
- Innovation award tracking
- Judgement Report PDF (print from browser)
- CSV export
- All data stored locally in browser

## Deploy to GitHub Pages

1. Create a new GitHub repo named `expo-judge`
2. Upload `index.html` to the repo root
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your app is live at `https://YOUR-USERNAME.github.io/expo-judge/`

## Usage
1. Fill in Judge Info (name, panel, expo name)
2. For each stall: enter details, score with stars, add feedback → Save
3. When done: **Session tab → End Judgement**
4. **Session tab → View & Download Judgement Report → Print / Save as PDF**
